# 最新版本 {#control-panel-releases}

本頁面列出了「控制面板」的所有新功能和改善項目。

## 2023年6月改進 {#june-2023}

**子網域的SSL憑證委派給Adobe**

您現在可以透過Adobe管理子網域的SSL憑證。 如果您是使用 CNAME 設定子網域，就會自動產生和提供憑證記錄，以便在您的網域託管解決方案中產生憑證。

請注意，只有在設定新的子網域時才可使用此功能。您無法為現有的委派子網域委派憑證。[了解更多](../subdomains-certificates/using/setting-up-new-subdomain.md)

>[!NOTE]
>
>Adobe管理的SSL是免費提供給使用者的功能。
